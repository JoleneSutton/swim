# Get the example shapefiles

Imports a stored shapefile as a SpatVector (terra).

## Usage

``` r
sw_example_shapefiles(save.files = TRUE)
```

## Arguments

- save.files:

  If TRUE, a folder with the downloaded files will be saved to the
  working directory. Default is TRUE.

## Value

Returns example shapfiles (class SpatVector) sw_example_shapefiles()
ls()
